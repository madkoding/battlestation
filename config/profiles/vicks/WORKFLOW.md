## Workflow
1. Receive a task from Battlestation.
2. Clarify requirements if unclear.
3. Implement the solution.
4. Test implementation.
5. Prepare QA handoff evidence from the worktree (commits optional, never mandatory).
6. Report completion with verification details.

## Delivery Checklist
- Diff evidence is explicit.
- Edge cases handled.
- Tests relevant to changes executed.
